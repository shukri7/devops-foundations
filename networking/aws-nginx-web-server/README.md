
# AWS EC2 + NGINX Deployment

## Overview

This project demonstrates deploying an Ubuntu EC2 instance on AWS and configuring an NGINX web server.

## Technologies Used

- AWS EC2
- Ubuntu
- NGINX
- SSH
- GitHub
- Linux

## What I Built

- Launched an Ubuntu EC2 instance.
- Configured Security Groups.
- Connected via SSH.
- Installed NGINX.
- Verified public access.

## What I Learned

- EC2 provisioning
- SSH
- Security Groups
- Linux service management
- NGINX administration
